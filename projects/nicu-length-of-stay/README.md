# Predicting NICU Length of Stay

## Overview
This project explores whether clinical and demographic variables available at NICU admission can predict neonatal length of stay.

## Data
The analysis uses the Neonatal Sepsis Care dataset containing neonatal demographic and clinical indicators recorded at admission.

## Methods
Three regression models were evaluated:
- Linear Regression
- Gradient Boosting
- XGBoost

## Key Findings
Admission-only variables had limited predictive power. Severity classification, gestational age, and birth weight showed the strongest influence.
