# Maternal Health Risk Prediction

**Skills:** Predictive Modeling | Healthcare Data Analysis | Risk Modeling

## Overview
This project develops predictive models to identify factors associated with maternal health risks using demographic and clinical pregnancy data.

## Dataset
Maternal health datasets containing clinical and demographic variables related to pregnancy.

## Methods
Machine learning models were used to evaluate relationships between maternal health indicators and risk outcomes.

## Key Findings
Several demographic and clinical variables were associated with increased maternal health risk.
